## [0.1.0] - 2026-02-19

### Added
- Initial release for UAT testing
- Project protection via FUSE overlay
- Automatic snapshot creation on file changes
- Snapshot open/save/cancel workflow
- File and folder restore from any snapshot
- Multi-agent workspace isolation
- Git integration with conflict detection
- Shell integration (bash, zsh, fish)
- `casso hawk setup` one-time configuration
- `casso hawk doctor` system diagnostics
- Blob storage with content-addressable deduplication
